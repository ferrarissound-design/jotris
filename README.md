# JOTRIS

HTML / CSS / JavaScript だけで作った、スマホ対応のテトリス風ブラウザゲームです。  
GitHub Pages でそのまま公開できます。

## 特徴

- 10列 × 20行の盤面
- 7種類のブロック
- 自動落下 / 左右移動 / 下移動 / 回転 / ハードドロップ
- ライン消去、スコア、レベルアップ、ゲームオーバー
- 次ブロック表示
- 最高スコア保存（`localStorage`）
- スマホ向け操作ボタン
- キーボード操作対応（PC）

## 操作方法

### PC
- ← → : 左右移動
- ↓ : 下移動
- ↑ or Space : 回転
- Enter : ハードドロップ
- R : リスタート

### スマホ
画面下のボタンで操作：
- ◀ 左移動
- ▶ 右移動
- ⟳ 回転
- ▼ 下移動
- ⤓ ハードドロップ

## ローカル実行

`index.html` をブラウザで開くだけで動作します。

## GitHub Pages 公開

1. このリポジトリを GitHub に push
2. Settings → Pages で公開ブランチを選択
3. 数分待つと公開 URL でプレイ可能
